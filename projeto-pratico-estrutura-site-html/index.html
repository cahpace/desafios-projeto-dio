O HTML

Voltar

HTML é uma linguagem de marcação, marcação de significado do texto. Ele dá semântica para o texto, por isso é importante saber sua estrutura geral e como aplicar a semântica corretamente, assim há melhor entendimento do navegador quando este for ler o código. Por isso é muito importante colocar as tags corretas, como tags de títulos para títulos, de parágrafos para parágrafos e assim por diante.

Estrutura básica HTML

Voltar

Praticamente todas as tags no HTML possuem uma estrutura igual, em que se tem a abertura e o fechamento dela. A abertura da tag é feita com os sinais de menor que e maior que, com o nome da tag entre eles. E o fechamento delas é feito com o sinal de menor que, barra, o nome da tag, e o maior que.

Tags, Atributos e Marcadores

Voltar

Apesar do HTML possuir muitas tags, não são todas que são muito utilizadas. Abaixo será apresentado as tags mais utilizadas, junto com seus atributos também mais utilizados, com uma breve descrição como utilizar.

  • h1, h2, h3, h4, h5, h6: são tags para títulos, sendo o h1 usado para o título principal e mais importante indo em ordem até o h6, que e o menos importante;
  • p: esse elemento é utilizada para marcar parágrafos;
  • a: é uma tag que funciona como uma âncora, ela liga o html com um outro documento por meio de um link (pode ser uma imagem, um arquivo, uma página em HTML, etc); Há alguns atributos que podem ser utilizados nesta tag. Por exemplo, ao colocar um link que direciona para uma página web, usa-se o atributo href para colocar o link, o atributo target que diz se a página será aberta na mesma aba ou se vai abrir em outra aba (com as opções self e blank, respectivamente), o atributo title para escrever o que aparece para o usuário quando este passa o mouse em cima da palavra com o link, e o nome do link antes de fechar a tag;
  • small: elemento utilizado para restrições legais, restrições de uso de conteúdo (como aquele símbolo de copyrights), etc;
  • i, u, strong, mark: são tags para deixar o texto itálico, sublinhado, em negrito e marcado como se fosse com marca texto, respectivamente;
  • sub, sup: são marcadores para deixar o texto sobrescrito (texto com letra menor e com alinhamento acima das letras do resto do conteúdo) e sobescrito (texto com letra menor e com alinhamento abaixo do resto do texto), respectivamente;
  • blackquote: é uma tag utilizada para citações longas, pois ele altera o alinhamento do texto em relação ao restante do conteúdo;
  • font: tag usada para alterar a fonte do texto;
  • del: é uma tag utilizada para identificar um texto excluído do documento, mas que mostra que havia texto, funcionando como histórico de modificações do documento;
  • abbr: é um elemento utilizado em abreviações e siglas dentro de um texto no html. Pode ser utilizado dentro de uma tag p;
  • iframe: é um elemento utilizado para incorporar páginas da web na página atual, como colocar um vídeo do Youtube na página atual, ou mesmo um mapa do Google Maps. Ao fazer isso, a vizualição do mapa, do vídeo, imagem de outras páginas é feita na página atual;

Textos

Voltar

Como o HTML têm função semântica, é importante que os textos sejam escritos dentro das tag adequadas: parágrafos dentro de tags p, abreviações dentro de tags abbr, etc.

Listas

Voltar

As listas podem ser ordenadas (cada item terá um número em ordem crescente), ou não ordenadas (cada item terá uma bolinha no lugar de um número). Para listas ordenadas, a tag utiliada é "ol" e para listas não ordenadas "ul". Dentro dessa tag, cada item deve ser colocado em uma tag "li", que marca da um deles.

Links

Voltar

Para colocar um link no HTML deve ser utilizado a tag "a" junto com o atributo "href". Nesse atributo terá o link que se quer abrir ao clicar (como o link para um site), e após o link, ainda dentro da tag "a" deve ser colocado a palavra ou nome com o link que aparecerá no navegador para ser clicado pelo usuário. Além do link levar para um site, ele também pode levar para alguma parte da mesma página aberta que esteja mais para baixo ou mais para cima. Como em um índice, que leva para uma parte mais abaixo da página e depois é possível subir para a parte inicial. Para isso, usa-se a tag "a" com "href" porém este não terá um link de site, mas sim um id. Assim, usa-se um hashtag junto ao nome do id, que levará à uma parte da página onde seja adicionado o atributo "id" com o nome do id com o hashtag.

Outro atributo que pode ser utilizado nas tags de links é o "target" e "title". No target, pode-se escolher a opção de "_self", que ao clicar no link abre a página na mesma aba, ou "_blank", que abrirá a página em outra aba. Apesar de originalmente a tag de link abrir na mesma aba, ao utilizar a opção "self" possibilita-se que, o usuário ao clicar com o botão direito em cima do link, consiga ter a opção de abrir em outra aba a página.
